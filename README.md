# Cloud François

> Le Cloud Souverain.

Site vitrine du projet **Cloud François**, un cloud souverain français. Hébergé via GitHub Pages.

## Déploiement

Ce site est conçu pour être déployé directement via **GitHub Pages** :

1. Créer un dépôt GitHub (ex: `cloud-francois`)
2. Pousser le contenu de ce dossier sur la branche `main`
3. Dans les paramètres du dépôt → Pages → Source : `main` / `/ (root)`
4. Le site sera accessible à `https://<username>.github.io/cloud-francois/`

## Structure

```
├── index.html      # Page principale
├── style.css       # Feuille de styles
├── script.js       # Interactions et animations
├── assets/
│   └── favicon.svg # Favicon du site
└── README.md       # Ce fichier
```

## Stack

- HTML5 sémantique
- CSS3 (variables, grid, flexbox, animations)
- JavaScript vanilla (IntersectionObserver, typing animation)
- Google Fonts (Inter, JetBrains Mono)
- Aucune dépendance, aucun framework
